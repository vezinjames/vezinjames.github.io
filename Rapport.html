Mon rapport

Voici toutes les choses que j'ai apprises au cours de mon apprentissage du HTML et du CSS:

-HTML

Pendant ma lecture du cours d'HTML, j'étais déjà enthousiaste à l'idée d'enfin savoir coder un site internet fonctionnel, ce qui fait que je m'y suis mis avec plaisir. J'en ai tiré des informations passionnantes, et j'ai pris beaucoup de plaisir à lire tout ceci.

J'ai tout d'abord compris que comme le dit le site developer.mozilla.org :
  • Le HTML n'est pas un langage de programmation à proprement parler, mais un langage de balises, composé d'éléments entourant le contenu.
  • Les éléments peuvent contenir des attributs donnant des informations supplémentaires sur l'élément sans pour autant l'afficher.
  • Au sein d'un élément, il est possible d'imbriquer un autre élément
  • Il peut y avoir des éléments appelés "vides", ils ne possèdent pas de contenu, on peut prendre l'exemple de l'élément < img >.
  • Un fichier HTML est organisé en plusieurs parties:
    • On définit d'abord le type de Document: < !DOCTYPE html >; ceci ne sert aujourd'hui que pour éviter les erreurs de script avec les anciens navigateurs.
    • L'élément < head > définit une sorte de boîte pour tout le contenu non affiché, et permet les liens avec les fichiers CSS notamment.
    • L'élément < body > est celui qui contient tout le contenu que l'on veut cette fois afficher, comme le texte par exemple.
  • Il peut contenir des images, soit imbriquées depuis l'ordinateur soit via un lien internet.
  • Et enfin il peut contenir des liens.

-CSS

Néanmoins, pour le CSS, je ne savais pas qu'il y avait un langage distinct pour le style du site, j'ai ensuite trouvé cela plus simple d'avoir procédé en utilisant deux langages. Cela permet effectivement de différencier telles lignes de codes pour le contenu de telles lignes de code pour le style.

Pareil que pour le HTML, le CSS n'est pas un langage de programation, mais cette fois un langage de feuille de style.

  • Il est composé:
    • D'un Sélecteur, qui va déterminer quelle partie du fichier HTML va être modifiée.
    • D'une Propriété, qui indique l'action que va faire la ligne.
    • D'une Valeur qui définit la Propriété.
    • Et le tout compose la Déclaration.
  • Il est possible d'imbriquer plusieurs éléments dans le Sélecteur.
  • Il existe plusieurs Sélecteurs ayant des foncions différentes.
  • Il est possible de choisir des fonts différents.
  • Il permet de changer le style d'une partie en particulier à sa guise.
